演習問題２

  • head要素
    • head要素とはページが読み込まれてもWebブラウザに 表示されない部分のとこを指す。
    例えば、＜title＞やCSSへのリンクの情報等の外部に見せたくないものを書く場所である。
    その代わり、headには文書のメタデータ(データを記述するデータ)が含まれる。大きなWebサイトではheadに多くの項目が入ることもある。
    今回は＜title＞要素と＜meta＞要素を用いた例と実行結果を示す。
  • マークアップ例
    •                     
                        <! DOCUTYPE html>
                        <html>
                           <head>
                             <meta charset="utf-8">
                             <title>演習問題２>
                           </head>
                           <body>
                             <p>これはテストです</p>
                           </body>
                        </html>

  • The h1,h2,h3,h4,h5 and h6要素
    • h1~h6の各要素はセクションの見出しのことを指す。各数字は見出しのレベルを表しており、数字が低いほどレベルが高いものとなっている。
    この見出しのレベルは文章のアウトラインに影響を与える。また、数値が等しければ文書内で同一レベルとして扱われる。
    以下に使用例と実行結果を示す。 ul>
  • マークアップ例
    •                 
                    <! DOCUTYPE html>
                    <html>
                       <head>
                         <meta charset="utf-8">
                         <title>演習問題２>
                       </head>
                       <body>
                         <h1 align="left">メインの見出し</h1>
                         <h2 align="left">通常の見出し</h2>
                         <h3 align="left">小見出し</h3>
                       </body>
                    </html>

  • section要素
    • section要素とは文書の自立した一般的なセクション(区間)を表す。ここにおけるsectionとは、コンテンツをテーマ別に分けたものである。
    文書における章や段落のことをさす。先ほど紹介したh1~h6の要素を用いて使用することが多い。
    以下に使用例と実行結果を示す。
  • マークアップ例
    •                 
                    <! DOCUTYPE html>
                    <html>
                       <head>
                         <meta charset="utf-8">
                         <title>演習問題２>
                       </head>
                       <body>
                         <section>
                             <h1> 見出しA</h1>
                             <p>一つ目のセクション </p>
                         </section>
                         <section>
                             <h1> 見出しA</h1>
                             <p> 二つ目のセクション</p>
                         </section>
                       </body>
                    </html>
